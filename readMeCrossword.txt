This program takes in input for a crossword puzzle.
The format file explains how the text file needs to be formatted.
The crossword.txt file is an example of actual input. To use the program your input needs to look like this.
To use the program the text file needs to be named crossword.txt and nothing else.
If double clicking the executable jar file named mjCrossWord.jar doesn't work.
Use the runCrossWord.bat file instead.
The crossword.txt file needs to be in the same folder as the mjCrossWord.jar for the program to work.
If you are using the runCrossWord.bat file it needs to be in the same folder as the mjCrossWord.jar file for it to work.
